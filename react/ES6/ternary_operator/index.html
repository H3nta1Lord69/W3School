React ES6 Ternary Operator

Ternary Operator

The ternary operator is a simplified conditional operator like if / else.

Syntax: condition ? <-expresion if true> : <-expresion if false>

Here is an example using if / else.

                    if (authenticated) 
                    {
                        renderApp();
                    } else 
                    {
                        renderLogin();
                    }

Try changing the "authenticated" variable in this file to false, and run the code to see what happens.

Here is the same example using a ternary operator:

                    authenticated ? renderApp() : renderLogin();

Try changing the "authenticated" variable in this file to false, and run the code to see what happens.
