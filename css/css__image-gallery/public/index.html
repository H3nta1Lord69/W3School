CSS Image Gallery

CSS can be used to create an image gallery.

Image Gallery

The following image gallery is created with CSS:

        div.gallery 
        {
            margin: 5px;
            border: 1px solid #ccc;
            float: left;
            width: 180px;
        }
          
        div.gallery:hover 
        {
            border: 1px solid #777;
        }
          
        div.gallery img 
        {
            width: 100%;
            height: auto;
        }
          
        div.desc 
        {
            padding: 15px;
            text-align: center;
        }
